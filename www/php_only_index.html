🌐 Introducción

Este servidor HTTP fue desarrollado en C++98 como parte del proyecto Webserv de 42. Soporta métodos GET, POST y DELETE, ejecución de CGI, subida de archivos y manejo de cookies y sesiones. En esta configuración se permiten tanto scripts PHP como Python.
